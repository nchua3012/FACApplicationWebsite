My Javascript Game

To practice what I've learnt about Javascript, I created a two-player fighting game. Play the game & view my Github repository.

Key Learnings

As a starting point, I followed a guide video by Chris Courses. Following his tutorial allowed me to set up the basic framework and implement the basic fighting game mechanics of movement and attacking, as well as setting up the win conditions with JavaScript.

Challenging aspects I encountered were:

  • Attack functions - Working on the various functions was particularly challenging as it required connecting the player inputs, collisions between hitboxes and character boxes as well as the correct sprites to be animated.
  • Sprite sheet positioning - Understanding the relationship between the character boxes/hit boxes and any offsets in X & Y positioning was also intially challenging and required a siginificant amount of time to solve. It required logging various positions as well as creating frames around all the seperate elements to debug all issues and create a functional system.

    Changes I've Made

    Once my basic framework was created, I made some changes:

  • Adding character facing directions according left or right inputs. This required the flipping of character sprites to the correct directions when a player inputs moving left or right.
  • Code to integrate more complex sprite sheets with multiple rows and columns for smoother character animations.

    • Looking ahead, I plan to make the following improvements as my understanding of Javascript continues to grow:

  • Move styling and layout to CSS for better organization.
  • Adjustments to the character code to allow easier positioning edits.
  • Add more attacks and blocking mechanics to enhance gameplay.
