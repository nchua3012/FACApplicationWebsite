My Javascript Game

To practice what I've learnt about Javascript, I created a game project. Play the game & view my Github repository.

Key Learnings

I chose to create a two-player fighting game as my game project. To get started, I followed a guide video by Chris Courses. Following his tutorial allowed me to set up the basic framework and implement the basic fighting game mechanics through JavaScript.

Key Javascript functions which I found challenging included:

  • Adding gravity
  • Working through hit registration via the interactions between hitboxes and characterboxes.

    Changes I've Made

    Once my basic framework was created, I added:

  • Correct movement directions for the characters when the players input moving left or right.
  • Code to integrate more complex sprite sheets with multiple rows and columns for smoother character animations.

    • Looking ahead, I plan to make the following improvements as my understanding of Javascript continues to grow:

  • Move styling and layout to CSS for better organization and scalability.
  • Adjustments to the character code to allow easier positioning edits
  • Add more attacks and blocking mechanics to enhance gameplay.
