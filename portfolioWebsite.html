My Portfolio Website

To help reinforce my understanding of HTML and CSS, I created a portfolio website as a hands-on project to practice the concepts I was learning.

Please have a look at my portfolio and find the Github repository.

Key Learnings

Some of the key topics I've learnt while creating the website included:

  • Implementing responsive features, such as a hamburger menu for smaller screens, to enhance the user experience across devices.
  • Styling the website using a CSS stylesheet to create a clean, visually appealing design.
  • Organizing and structuring my code efficiently, ensuring clear and readable sections that make future updates and changes easier to manage.

    This project has been a valuable learning experience, and I will continue to update it as I refine my skills and expand my knowledge in development.
